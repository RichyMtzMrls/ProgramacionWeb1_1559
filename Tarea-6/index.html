Lenguaje de Marcado de Hipertexto (HTML)

Bienvenidos! Hoy estaremos practicando algunos conceptos y elementos más importantes vistos hasta el momento en la materia de pw1, además veremos el uso de etiquetas semánticas en la estructuración de un sitio web.

Conceptos Principales de HTML

HTML

Lenguaje de Hipertexto, que se utiliza para hacer y estructurar una págin web.

Elemento HTML

Es la unidad basica de estructura y contenido de una pagina web, esta compuesta de:

  1. Etiqueta de Apertura
  2. Pares Atributo-Valor
  3. Contenido
  4. Etiqueta de Cierre

Etiqueta <head>

Contiene todo el contenido HTML

Contenendores Semánticos

<header> Etiqueta utilizada para el contenido introductorio de una pagina web, puede contener el logo, el menú de navegación principal, el nombre de la empresa, etc...
<nav> Etiqueta utilizada para el menú de navegación principal, suele contener enlaces a otras paginas relacionadas con navegar por la página principal, no debe haber más de uno en la página.
<main> Etiqueta única de una página web cuyo propósito es resaltar nuestro objetivo o mensaje principal del sitio web, dentro se colocara el contenido principal de nuestra página.
<section> Es una etiqueta que poco a poco se intenta que se sustituya a <div>, su uso es dividir la página en secciones y agrupar contenido relacionado.
<article> Es una etiqueta que se utiliza para agrupar contenido autonomo que si se separa de otro contenido sigue teniendo coherencia, normalmente se compone de un encabezado y un contenido.
<footer> Es una etiqueta que normalmente se encuentra hasta el final de una página y funciona como cierre del sitio web, a menudo contiene datos de contacto, derechos de autor u otra información complementaria.

Etiquetas Principales HTML

Etiqueta <a>

Es un elemento que nos va a permetir navegar a:

  • Páginas Externas
  • Páginas Internas
  • Secciones dentro de nuestra misma página (enlaces Ancla).

Algunos valores específicos para el atributo href son:

href="https://wa.me/553246197"
Abre un chat del número especificado en Whatsapp.
href="mailto:example@a.com"
Abre una aplicación de correo, con el destinatario especificado.

Etiqueta <form>

Es el medio con el que los usuarios pueden comunicarse con nuestra página web, atraves de este elemento podemos recopilar información para posteriormente procesarla
